## Descripcion del proyecto
```Sitio web desarrollado en reactjs destinado a la venta de celulares de diversas marcas, utilizando firebase como base de datos simulada.```
## Contenido del proyecto
* Componente Navbar con links a distintos tipos de productos, utilizando routes.
* Componente ItemList para mostrar los productos con distintos metodos de array y promesas
* Componente Item detail para mostrar mas detalles del producto.
* Componente Cart para mostrar los productos seleccionados y calcular el monto de la compra; con funciones como limpiar cart o remover item.
* Componente CartContext para acceder a distinta informacion y funciones en comun desde todos los componentes
* Uso de varias herramientas de React como Hooks (useState y useEffect)
* Uso de firebase para simular base de datos e interactuar con el front
## Lenguajes y metodologias utilizadas
* HTML y CSS
* JavaScript
* React
* Bootstrap
* Sweetalerts
* Google Firebase
